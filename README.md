TicTacToe
=========

Simple tic tac toe simulator in __java__.

To run it just use `sh play` or `./play` commands.

__Sample output__

    Gioca il giocatore X. Inserisci le coordinate (riga,colonna) 
    1,1

     | | 
    ------
     | | 
     
    Gioca il giocatore O. Inserisci le coordinate (riga,colonna) 
    2,2

     |O|

     | |
